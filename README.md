# Environmental_Monitoring_System
A system to monitor the environment; temperature, humidity, pressure.

Embedded systems project using an STM32 Nucleo-F446RE.

# Goals

- Learn STM32 development
- Learn GPIO, UART, and I2C
- Interface with a BME280 sensor
- Display data on an SSD1306 OLED
- Implement FreeRTOS multitasking

# Hardware

- STM32 Nucleo-F446RE
- BME280
- SSD1306 OLED

# Status
